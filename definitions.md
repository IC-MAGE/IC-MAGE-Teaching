# Notation
* Ram pressure = R
* Cell pressure = C
* Back pressure = B
* Deviatoric stress = q
* Mean effective stress = $p'$
* Axial effective stress = $\sigma'_a$
* Radial effective stress = $\sigma'_r$
* Change in pore water pressure = $du$ (for drained tests, this is always zero)

> Initial values (i.e. thse at the start of a test, i.e. on the first row) have a subscript 0, i.e. start of test back pressure is $B_0$

# Definitions / relationships
* $R=q$
* $B=B_0+du$
* $C=B+p'-\frac{q}{3}$

> We can set $B_0=100$ kPa (as an arbitrary reference value).
